# Terraform Local Practice

I originally started learning Terraform with Azure. However, I wanted to continue practicing Terraform fundamentals even without deploying cloud resources.

This folder contains local Terraform projects that help me understand how Terraform works independently of any cloud provider.

The exercises focus on concepts such as:

- Providers
- Resources
- Variables
- Outputs
- State
- Modules

Once these concepts are understood, they can be applied to cloud providers such as Azure or AWS.
